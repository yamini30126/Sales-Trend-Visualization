# Sales Trend Visualization

## Overview
This project analyzes the Superstore Sales dataset using Python and visualizes sales trends across months, years, product categories, and regions.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Dataset
- **Name:** Superstore Dataset
- **Source:** Download dataset from Kaggle: https://www.kaggle.com/datasets/vivek468/superstore-dataset-final
- **File:** Sample - Superstore.csv
- **Records:** 9,994 rows | 21 columns
- **Total Sales:** $2,297,200.86

## Project Description
The dataset contains sales information such as order date, product category, region, sales amount, quantity, discount, and profit. The notebook performs data cleaning, analysis, and visualization to identify key sales trends and business insights.

## Visualizations
1. Monthly Sales Trend - Line graph showing sales across all 12 months
2. Yearly Sales Trend - Bar chart showing year-over-year sales (2014-2017)
3. Category wise Sales - Bar chart comparing Furniture, Office Supplies, and Technology
4. Region wise Sales - Bar chart comparing Central, East, South, and West regions

## Key Findings
| Finding | Result |
|---|---|
| Best Month | November |
| Best Year | 2017 |
| Top Category | Technology |
| Top Region | West |

## How to Run
1. Open the notebook in Google Colab
2. Upload the Sample - Superstore.csv dataset
